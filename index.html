Gate Information

  • ℹ️ AND: Outputs TRUE only if both inputs are TRUE.
  • ℹ️ OR: Outputs TRUE if at least one input is TRUE.
  • ℹ️ NOT: Inverts the input signal (TRUE → FALSE, FALSE → TRUE).
  • ℹ️ XOR: Outputs TRUE if only one input is TRUE.
  • ℹ️ SWITCH: Manually toggles between TRUE (ON) and FALSE (OFF).
  • ℹ️ BULB: Lights up if it receives a TRUE signal.

How to Use

  • Drag gates and components from the left panel to the canvas.
  • Click and drag between gates to create connections.
  • Click the switch to toggle its state.
  • Reposition components by dragging them within the canvas.
